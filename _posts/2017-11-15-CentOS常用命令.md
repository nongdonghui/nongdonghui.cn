1.解压

```
解压gz文件
tar zxvf /app/server.tar.gz -C /app/server

解压war文件
unzip /app/server.war -d /app/server
```

2.查看磁盘空间剩余量

```
df -h
```

3.删除日志文件

```
sudo find /var/log/ -type f -mtime +30 -exec rm -f {} \;
```

4.十进制转换为16进制

printf "%x\n" 12979
